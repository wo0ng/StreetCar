# StreetCar
encountering cars on the street
```
[문제 설명]
여러 대의 차가 동쪽과 서쪽으로 달리고 있습니다.
동쪽으로 달리는 차와 서쪽으로 달리는 두 차가 서로 스쳐서 지나가는 모든 횟수를 구하세요.
동쪽으로 달리는 차는 0, 서쪽으로 달리는 차는 1이라 했을 때, 현재 도로 위에 있는 차의 정보가 담긴 배열 cars가 매개변수로 주어집니다. 스친 횟수를 return 하도록 solution 함수를 완성해주세요

[문제 조건]
- 0 <= cars의 길이 <= 100,000
- cars의 모든 원소는 0 또는 1입니다.

[입출력 예]
입력: [0,1,0,1,1]
출력: 5

입력: [1,0,0,0,1,0]
출력: 3

입력: [1,1,1,1,0,0]
출력: 0

입력: [1]
출력: 0

[입출력 설명]
배열이 [0,1,0,1,1]로 주어졌다면
0->     0->
    <-1     <-1    <-1

과 같습니다. 따라서 차들은 총 5번 스쳐지나갑니다.
```
